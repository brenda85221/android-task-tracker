# TaskTracker - 簡單任務管理 App

## 專案簡介
TaskTracker 幫助你快速新增、分類、追蹤任務，支援任務圖片、分類顏色標記與通知提醒（開發中）。  
此 App 也可作為優惠券、折價券整理庫或日常待辦事項管理工具。

## 功能
- 新增 / 編輯 / 刪除任務
- 任務種類分類與顏色標記
- 任務圖片新增與預覽
- 依任務種類與月份進行分組顯示
- 左滑快速刪除任務
- 任務完成狀態切換
- 通知提醒功能（開發中）

## 技術
- Kotlin / Android SDK
- MVVM 架構
- Room Database：任務與任務種類資料管理
- RecyclerView + ViewBinding：任務列表顯示
- Photo Picker API：任務圖片選擇
- Extension Functions：時間格式化與工具函式
